## 개발자 서상혁

웹 에이전시를 직접 운영하며 AI 도구를 도입해 기존의 비효율적인 개발 프로세스를 혁신했던 경험은, 제게 기술이 비즈니스에 적용될 때 만들어내는 '실질적인 효율'을 체감하게 했습니다. 이 경험을 계기로 단순히 기능을 구현하는 것을 넘어, 최적의 방법으로 문제를 해결하는 개발자가 되기를 결심했습니다.

복잡한 코드를 자랑하는 것이 아니라, 사용자의 불편함을 가장 효과적으로 해소하는 것이라 생각합니다. 이를 바탕으로, 100명 규모의 인력 관리 업무를 QR 시스템으로 자동화하여 리소스를 획기적으로 절감하고, RAG 챗봇 개발 시 상황에 맞는 기술 스택 변경으로 프로젝트를 완주하며 '기술을 통한 문제 해결'을 실천해 왔습니다.

앞으로도 특정 도구에 얽매이지 않고 문제의 본질을 꿰뚫는 개발자로 성장하고 싶습니다. 끊임없는 학습과 실행력을 바탕으로, 사용자와 비즈니스 모두에게 확실한 가치를 전달하는 서비스를 만들어가겠습니다.
